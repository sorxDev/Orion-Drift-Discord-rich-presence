# A2 Station — Discord Rich Presence

Shows a live A2 station on your Discord profile.

## Setup

### 1. Install A2 Station Presence Setup 2.0.0
or
### 1.5. Open A2-Station-Presence-portable

### 2. Set your Api Key

Get it from: https://dashboard.oriondrift.net → https://dashboard.oriondrift.net/account → Create Key

### 3. Enjoy!

## What should show on your profile
| Field | Value |
|---|---|
| Large image | Orion Drift Icon |
| Small image | Fleet Icon |
| Line 1 | Station name |
| Line 2 | `42 / 75 players` (live) |
| Elapsed time | How long since you selected the station |

Have fun showing what station your in!
